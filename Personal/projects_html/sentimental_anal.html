Sentimental Analysis on Tweet's

Project information

  • Category: Web design , Machine Learning
  • Client: None(Personal)
  • Project date: 10 July, 2021
  • Project URL: Thiruvikramchoudry/Sentimental_analysis_twitterHash

I have build a user interface web application to analyse a group of tweets and conclude the sentiment of it . This web page will also provide some graphical representation of the tweets based on their sentiment and give more information about the hashtag name. This website generally contains two pages , the home page and the tweet analysis page . When a user enters the home page ,the page will ask them to enter a hashtag name . After entering , the hashtag name will move to the server site where sentiment analysis takes place based on the entered name. Finally a models will be created and it conclude whether the tweet is positive,negative or neutral and dispaly's in the webpage.
